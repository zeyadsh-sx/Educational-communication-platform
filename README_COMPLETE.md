# منصة الاتصال التعليمي - Educational Communication Platform

## نظرة عامة

منصة تعليمية متكاملة مصممة لتسهيل التواصل بين الطلاب وأساتذتهم في بيئة تعليمية آمنة وفعالة. المنصة تدعم اللغة العربية بشكل كامل وتوفر واجهة سهلة الاستخدام.

## المميزات الرئيسية

### للأساتذة (Professors)
- 👨‍🏫 إنشاء وإدارة الكورسات الدراسية
- 📤 رفع المواد الدراسية والمحاضرات
- 💬 الإجابة على أسئلة الطلاب
- 📢 نشر الإعلانات والتواصيل
- 📅 إدارة المواعيد المكتبية
- 📊 عرض إحصائيات وتقارير الأداء

### للطلاب (Students)
- 👨‍🎓 التسجيل في الكورسات المتاحة
- 📥 الوصول للمواد الدراسية وتحميلها
- ❓ طرح الأسئلة والحصول على إجابات
- 📅 حجز المواعيد المكتبية
- 🔔 متابعة الإعلانات والتحديثات
- 📈 عرض التقدم الأكاديمي

### عامة (General)
- 🔐 نظام مصادق آمن (Authentication)
- 🌍 واجهة عربية بالكامل
- 📱 تصميم متجاوب (Responsive Design)
- 🛡️ حماية البيانات والملفات
- 🔔 نظام إشعارات فعال
- 🎨 ايموجيز واقعية باستخدام Twemoji

## قاعدة البيانات

المنصة تتطلب قاعدة بيانات MySQL بالجداول التالية:

- `users` - معلومات المستخدمين (طلاب وأساتذة)
- `courses` - معلومات الكورسات
- `materials` - المواد الدراسية
- `questions` - أسئلة الطلاب
- `appointments` - المواعيد المكتبية
- `announcements` - الإعلانات
- `enrollments` - تسجيل الطلاب في الكورسات

## التثبيت

### المتطلبات
- PHP 7.4 أو أعلى
- MySQL 5.7 أو أعلى
- Apache أو Nginx
- ملحقات PHP: mysqli, session, fileinfo

### خطوات التثبيت

1. **تحميل المشروع**
   ```bash
   git clone <repository-url>
   cd Educational-communication-platform
   ```

2. **إعداد قاعدة البيانات**
   ```sql
   CREATE DATABASE educational_platform;
   -- استيراد ملف database_schema.sql
   mysql educational_platform < database_schema.sql
   ```

3. **إعدادات الاتصال**
   - تعديل `config/database.php` ببيانات الاتصال بقاعدة البيانات
   - تعديل `config/settings.php` بالإعدادات المطلوبة

4. **المجلدات المطلوبة**
   ```bash
   mkdir -p uploads/materials
   chmod 777 uploads
   ```

5. **الوصول للمنصة**
   - فتح المتصفح على `http://localhost/Educational-communication-platform`
   - إنشاء حساب أستاذ أو طالب

## استخدام المنصة

### إنشاء حساب أستاذ
1. اضغط على "إنشاء حساب"
2. اختر "دكتور" كنوع الحساب
3. املأ المعلومات المطلوبة
4. سجل دخولك وابدأ بإنشاء الكورسات

### إنشاء حساب طالب
1. اضغط على "إنشاء حساب"
2. اختر "طالب" كنوع الحساب
3. املأ المعلومات المطلوبة
4. سجل دخولك وانضم للكورسات المتاحة

## تحسينات الايموجيز (Emoji Improvements)

### استخدام Twemoji
تم تحديث المنصة لاستخدام **Twemoji** - مكتبة من Google لعرض ايموجيز واقعية وموحدة عبر جميع الأنظمة والمتصفحات.

### ملف Helper الجديد
- **`includes/emoji.php`**: مكتبة دوال مساعدة للعمل مع الايموجيز
  - `getEmoji($key)`: الحصول على ايموجي معين
  - `emojiSpan($emoji)`: إنشاء span للايموجي
  - `emojiWithText($emoji, $text)`: دمج الايموجي مع النص
  - `emojiBadge($text, $emoji)`: إنشاء badge مع ايموجي
  - `getAchievementBadge()`: عرض شارات الإنجاز
